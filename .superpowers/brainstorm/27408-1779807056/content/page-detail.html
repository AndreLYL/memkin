Memoark is a local-first personal memory storage backend using PGLite + pgvector + Hono + MCP SDK.

It implements hybrid search combining full-text search (PostgreSQL tsvector) with vector similarity search using reciprocal rank fusion (RRF).

The system supports knowledge graph traversal via BFS with configurable depth, timeline entries, and tag-based organization.
